A project designed to monitor and manage call queues. In this project, I used Apollo/GraphQL to connect to GraphQL via WebSocket in real time, retrieving and displaying calls. Later, I sent actions such as pause, unpause, and spy to manage the calls.

Tech stack: Reactjs, Typescript, Apollo/Graphql
