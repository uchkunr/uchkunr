### Hey, I'm Uchkun Rakhimov 👋

### About me

I build systems that keep people and data in sync from scalable back-end services on NestJS and PostgreSQL, to fast, reactive front-ends with React and Vite. I enjoy working in monorepos and automating deployments with Docker, because it makes development smoother for the whole team. For me, technologies are just tools the real challenge (and fun) is making them work together reliably at scale, whether it’s handling thousands of requests per day or keeping real-time features running without a glitch.
